# Sec-Toolkit · 轻量级安全工程化工具集

欢迎来到 Sec-Toolkit 组织！

## 项目列表

| 项目 | 语言 | 描述 |
| --- | --- | --- |
| [dirscan-toolkit](https://github.com/sec-toolkit/dirscan-toolkit) | Python | 异步目录扫描器，支持 SARIF 和 JSON 输出 |
| [portscan-toolkit)](https://github.com/sec-toolkit/portscan-toolkit) | Go | 高速端口扫描器，支持服务指纹识别 |
| [reporter](https://github.com/sec-toolkit/reporter) | Python | 统一扫描报告生成器，支持 HTML 和 SARIF 格式 |
| [pipelines](https://github.com/sec-toolkit/pipelines) | YAML | DevSecOps CI/CD 模板，支持 GitHub Actions |
| [elk-suricata](https://github.com/sec-toolkit/elk-suricata) | Docker | 一键部署 ELK + Suricata，支持流量分析和告警 |
